# Poetrax

Музыкальная платформа для стихов.

## Версия 1.0.0

Релиз: 22 марта 2026

### Возможности
- 🎵 Прослушивание треков
- 📖 Чтение стихов
- 👤 Профили поэтов
- ❤️ Лайки и закладки
- 🔍 Поиск по всему контенту
- 📱 Адаптивный дизайн
- 💎 Персонализированные рекомендации

### Технологии
- PHP 8.2
- PostgreSQL 15
- Redis 7
- Vanilla JS

### Установка
1. Скопировать файлы
2. Настроить `.env`
3. Запустить миграции
4. Готово!
